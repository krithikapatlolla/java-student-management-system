# Student Management System

A Java-based console application used to manage student records efficiently.

## Features
- Add Student
- View Students
- Search Student By ID
- Search Student By Name
- Update Student Details
- Delete Student
- Grade Calculation
- File Handling

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- File Handling

## How To Run
1. Compile the Java file
2. Run the program
3. Use menu options to manage students
